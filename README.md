# 🚀 Job Portal - Full Stack MERN Project

A full-featured Job Portal web application built using the MERN stack (MongoDB, Express, React, Node.js). This platform allows users to register, login, browse jobs, apply for jobs, and allows companies to post and manage job listings.

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite / CRA)
- Axios
- React Router DOM
- Context API / Redux (if used)
- Tailwind CSS / CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt.js
- Cookie Parser
- CORS

---

## 📌 Features

### 👤 User Features
- User registration & login
- JWT authentication (secure login)
- Browse all available jobs
- Search jobs by title/location
- View job details
- Apply for jobs
- View applied jobs
- Logout system

---

### 🏢 Recruiter Features
- Company registration/login
- Post new job listings
- Update job details
- Delete job posts
- View applicants for each job

---

## 🧠 Core Functionalities

- 🔐 Secure authentication using JWT
- 🍪 HTTP-only cookies for session management
- 🗂️ RESTful API architecture
- 🔄 CRUD operations (Jobs, Users, Applications)
- 🌐 Cross-Origin Resource Sharing (CORS enabled)
- ☁️ MongoDB Atlas cloud database integration
- 📡 Axios-based API communication
- ⚡ Responsive UI for all devices
